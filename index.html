Before you open it

Version 0.1.0 is ad-hoc signed and not notarized, so macOS may show a Gatekeeper warning on first launch.

Good to know.

Which macOS versions are supported?

macOS 15 or later on Apple silicon and Intel Macs.

Where does usage data come from?

Integrations may read local CLI history, configuration, authentication state, Keychain, SQLite, or JSON. Some contact provider services. Version 0.1.0 is not sandboxed. See the privacy notes.

Is source code included?

No. This repository contains product documentation and prebuilt releases.
