# TokenUsage 0.1.0

The first binary release of TokenUsage, a native macOS menu bar app for keeping
AI usage limits and token activity in sight.

## Highlights

- Integrations for Antigravity, Claude, Codex, GitHub Copilot, Cursor, Devin,
  Grok, OpenRouter, and Z.AI
- Local token statistics for Claude, Codex, Cursor, and Grok
- 30-day breakdowns by provider and model
- Estimated costs when model pricing is available
- Customizable menu bar metrics and ordering
- Usage-pacing and low-limit notifications
- Automatic refresh, global shortcut, and launch at login
- Native macOS widgets
- English, Simplified Chinese, and Japanese interfaces

## Requirements

- macOS 15 or later
- Apple silicon or Intel Mac

## Install

Download `TokenUsage-macOS-universal.zip`, extract `TokenUsage.app`, and move it
to Applications. On first launch, Control-click the app, choose **Open**, and
review the macOS warning before continuing.

## Signing notice

This build uses an ad-hoc code signature and is not notarized. macOS may show a
Gatekeeper warning on first launch.

## Distribution note

This release contains the prebuilt application. Source code is not included in
this repository.
