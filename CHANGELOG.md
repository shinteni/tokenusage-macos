# Changelog

## 0.1.0 — 2026-07-14

First binary release.

- Added integrations for Antigravity, Claude, Codex, GitHub Copilot, Cursor,
  Devin, Grok, OpenRouter, and Z.AI

- Added 30-day provider and model breakdowns with estimated costs
- Added configurable menu bar metrics, notifications, automatic refresh, and a
  global shortcut
- Added native macOS widgets
- Added English, Simplified Chinese, and Japanese interfaces
